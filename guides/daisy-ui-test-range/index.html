DaisyUI Test Range

This test range mainly serves the purpose of testing your current theme against the built-in components of DaisyUI. It includes a variety of components such as buttons, alerts, badges, and more. If it works well with the components in this test range, it should work well with this template.

Daisy generator components

As seen on the DaisyUI documentation, you can use any of the built-in components or create your own custom components.
