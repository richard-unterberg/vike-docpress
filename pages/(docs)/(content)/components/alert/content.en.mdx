# Alert Component

The Alert component is used to display important messages or warnings to the user. It can be used to highlight critical information, errors, or any other type of message that requires the user's attention.

## Basic Usage

<Presenter>
  <Alert>
    Hello I'm the default alert, which is of type `info`. More info to type, see [below](#alert-types).
  </Alert>
</Presenter>

<CodeGroup>
```mdx
import { Alert } from '@/components/docs'

<Alert>
  Hello I'm the default alert, which is of type `info`. More info to type, see [below](#alert-types).
</Alert>
```

```tsx
import { Alert } from '@/components/docs'

export const MyComponent = () => {
  return (
    <div>
      <Alert>
        Hello I'm the default alert, which is of type `info`. More info to type, see <a href="#alert-types">below</a>.
      </Alert>
    </div>
  )
}
```
</CodeGroup>

## Alert Types

The following types of alerts are available: `success`, `error`, `warning`, and `info`. Each type has its own styling and icon to visually differentiate the importance or nature of the message.

<Presenter>
  <Alert type='error'>
    I'm an error alert, indicating a problem or issue that needs attention.
  </Alert>
  <Alert type='warning'>
    I'm a warning alert, indicating a potential issue or cautionary message.
  </Alert>
  <Alert type='success'>
    I'm a success alert, indicating a positive action or outcome.
  </Alert>
  <Alert type='info'>
    I'm an info alert, indicating general information or a neutral message.
  </Alert>
</Presenter>

<CodeGroup>


## Custom Content

The Alert component can contain any custom content, including headings, paragraphs, lists, and more. This allows you to create rich and informative alerts that can effectively communicate your message to the user.

<Presenter>
<Alert type='success'>
  ### Custom Alert Content
  **I'm a success alert,** <br /> 
  indicating a positive action or outcome.
  - hey
  - I'm an error alert, indicating a problem or issue that needs attention.
</Alert>
</Presenter>

<CodeGroup>
```mdx
import { Alert } from '@/components/docs'

<Alert type='success'>
  ### Hey
  **I'm a success alert,** <br /> 
  indicating a positive action or outcome.
  - hey
  - I'm an error alert, indicating a problem or issue that needs attention.
</Alert>
```

```tsx
import { Alert } from '@/components/docs'

export const MyComponent = () => {
  return (
    <div>
      <Alert type='success'>
        <h3>Hey</h3>
        <p><strong>I'm a success alert,</strong> <br /> indicating a positive action or outcome.</p>
        <ul>
          <li>hey</li>
          <li>I'm an error alert, indicating a problem or issue that needs attention.</li>
        </ul>
      </Alert>
    </div>
  )
}
```
</CodeGroup>
